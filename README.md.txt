# CodeAlpha_IrisClassification

This project is part of the CodeAlpha Data Science Internship.

## Objective
Classify Iris flower species (Setosa, Versicolor, Virginica) using machine learning.

## Tools Used
- Python
- Scikit-learn
- Pandas
- Seaborn
- Matplotlib

## Steps
1. Load the Iris dataset.
2. Visualize relationships between features.
3. Train a Logistic Regression model.
4. Evaluate accuracy and performance.

## Results
Achieved ~97% accuracy on test data.
